Phexum NLP Platform

Design & Development

Phexum is a company that provides artificial intelligence and natural language processing solutions. It is aimed to serve the company's NLP platform as a Saas product. Jira was preferred for the project management of the team consisting of 4 people. Git was used for the version control of the source code. The backend module was developed based on the principles of Multi Instance and Dependency Injection, using the Java11-SpringBoot framework and the Maven tool. Hexagonal Architecture was preferred in order to minimize the dependency of the application on the technologies in the system. In the frontend module, VueJs- VueX / Bootstrap4.x is preferred. Due to the formation of relational entities, PostgreSql was preferred on the database side and JPA was preferred on the ORM side. Restful services have been developed so that the frontend module can access backend resources over http and users can consume the provided APIs. It is designed as stateless to provide load balance infrastructure and no data is kept in JVM memories. Therefore, Redis was used to manage common states in application inctances. The project was initialized and deployed with Docker.
